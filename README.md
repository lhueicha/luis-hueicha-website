# Luis Hueicha - Web Personal

Web personal de Luis Hueicha - Ingeniero Civil en Obras Civiles • Máster en Ingeniería Geotécnica

## Enlaces

- **GitHub:** https://github.com/luishueicha/luis-hueicha-website
- **Deploy:** https://luis-hueicha.vercel.app (o tu dominio)

## Features

- Diseño moderno y responsivo
- Hero section con gradientes
- Sobre mí, proyectos, habilidades, contacto
- Next.js 14 + Tailwind CSS
- SVG icons nativos (sin dependencias externas)

## Estructura

```
luis-hueicha-website/
├── src/
│   └── app/
│       ├── page.tsx       # Web principal
│       └── globals.css    # Estilos globales
├── package.json
├── next.config.js
├── tsconfig.json
└── postcss.config.js
```

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deploy

1. Crea un repo en GitHub: `luis-hueicha-website`
2. Sube los archivos
3. Deploya en Vercel o tu hosting favorito
